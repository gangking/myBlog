let result=arr.reduce(function (tmp, item, index){
    //alert(tmp+','+item+','+index);

    return tmp+item;// 总和=当前的临时结果+这次的结果

# generator-生成器

## 本质：

```
一种特殊函数，普通函数一条路走到黑，一旦开始就直到结束
而generator中间能停
```

## 案例：

```
// 创建一个generator
function *show(){
      alert('a');
      yield;// 暂时放弃执行
      alert('b');
}
let genObj=show();// 用法
genObj.next();  
genObj.next();  
```

## 总结：

踹一脚走一步

以下是本质：

```
// 创建一个generator
function *show(){
      alert('a');
      yield;// 暂时放弃执行
      alert('b');
}
// 相当于创建了两个函数show_1，show_2
function show_1(){
      alert('a');
}
function show_2(){
      alert('b');
}
//
let genObj=show();// 用法
genObj.next();    //show_1
genObj.next();    //show_2
```



## 意义：

```
1、正常函数（异步只能回调）
function 函数(){
  代码...
  ajax(xxx, function (){
    代码...
  });
}
2、generator
function *函数(){
  代码...
  yield ajax(xxx);// 暂停的时候配置数据操作
  代码...
}

也没有巨大区别，
就是能走走停停
```

# generator- yield是啥？

```
function *炒菜(菜市场买回来的){
      洗菜->洗好的菜

      干净的菜->切->丝

      let 切好的菜=yield 丝;

      切好的菜->炒->熟的菜

      return 熟的菜;
    }
```
